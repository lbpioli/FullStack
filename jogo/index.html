Sobre o jogo

"Caça aos Vírus" é um jogo onde você é um antivírus digital. Sua missão é eliminar vírus que tentam invadir o sistema antes que causem danos!

Clique para atirar nos vírus antes que escapem. Cada acerto dá pontos. Cuidado: se muitos escaparem, o sistema será infectado!

Projeto desenvolvido para a disciplina de Desenvolvimento Web
